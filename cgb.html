What is Scan Conversion? [2 Marks]

Scan Conversion ek process hai jisme graphical objects ko raster format (pixels) mein convert kiya jata hai taaki unhe display screens par dikhaya ja sake. Yeh process vector graphics ko pixels mein convert karke image quality ko maintain karta hai, jisse high-resolution images display hoti hain.

What is Rasterization? [2 Marks]

Rasterization ek technique hai jisme vector graphics (jisse mathematical equations se define kiya jata hai) ko pixels mein convert kiya jata hai. Is process mein har pixel ka color set kiya jata hai, jo ultimately screen par visible image banata hai. Yeh process computer graphics ka ek core component hai.

Define Clipping and Clip Window. [2 Marks]

Clipping ek process hai jisme graphical objects ko display window ke boundaries ke andar restrict kiya jata hai. Isse unwanted portions ko remove kiya jata hai. Clip window wo area hai jahan par graphics ko dikhana hota hai. Clipping se rendering efficiency badh jaati hai aur unwanted computations se bachne mein madad milti hai.

Distinguish between Window Port and Viewport? [2 Marks]

Window Port ek rectangular area hai jahan se graphical objects ko clip kiya jata hai, jabki Viewport wo area hai jahan clipped image display hota hai. Window Port graphics ko define karta hai, jabki Viewport unhe screen par kaise dikhaya jata hai yeh define karta hai. In dono ka combination efficient rendering aur display control ke liye zaroori hai.

What is the importance of 8-way symmetry in scan conversion of circle? [2 Marks]

8-way symmetry ka matlab hai ki agar circle ke ek quadrant ko calculate kiya jaye, toh baaki ke quadrants bhi usi symmetry ka use karke calculate kiye ja sakte hain. Isse computational efficiency badh jaati hai, kyunki ek hi quadrant ki calculation se poore circle ka rendering ho sakta hai. Yeh method circle ke liye bahut effective hai, jo performance ko enhance karta hai.

Differentiate between DDA and Bresenham's line drawing algorithm. [7 Marks]

DDA (Digital Differential Analyzer) aur Bresenham's algorithm dono line drawing algorithms hain, par inke kaam karne ke tarike alag hain. DDA algorithm floating point arithmetic ka use karta hai, jabki Bresenham’s algorithm integer arithmetic ka use karta hai, jo use efficient aur faster banata hai. DDA mein round-off errors ka risk hota hai, jabki Bresenham's algorithm yeh issue avoid karta hai. DDA ko simple implementation ke liye jaana jata hai, jabki Bresenham's ko accurate line drawing ke liye prefer kiya jata hai.

Explain Bresenham’s line drawing algorithm. Draw a line from (2,3) to (5,8) by using Bresenham’s algorithm. [7 Marks]

Bresenham’s line drawing algorithm ek efficient method hai jo integer arithmetic ka use karke line draw karta hai. Yeh algorithm yeh decide karta hai ki agla pixel kis position par plot karna hai, jisse line draw hoti hai.

Steps to draw line from (2,3) to (5,8):

  • Calculate dx = x2 - x1 = 5 - 2 = 3
  • Calculate dy = y2 - y1 = 8 - 3 = 5
  • Calculate initial decision parameter p = 2dy - dx = 2(5) - 3 = 7
  • For each x from x1 to x2:
  • If p < 0, then plot (x, y) and increment x.
  • If p >= 0, then plot (x, y) and increment x and y.

Is tarike se, Bresenham’s algorithm use karke line draw ki ja sakti hai.

Write DDA algorithm and also write the advantage and disadvantage of this algorithm. [7 Marks]

DDA algorithm ek simple line drawing technique hai jo pixels ko sequentially compute karta hai. Iska formula hota hai:

DDA Algorithm:

  1. Start with two endpoints (x1, y1) and (x2, y2).
  2. Calculate dx = x2 - x1 and dy = y2 - y1.
  3. Calculate steps = max(|dx|, |dy|).
  4. Calculate x increment = dx / steps and y increment = dy / steps.
  5. Plot the pixels at (x, y) for each step.

Advantages:

  • Simpler implementation.
  • Good for generating lines with variable slopes.

Disadvantages:

  • Less efficient due to floating-point calculations.
  • Round-off errors can occur.

Write midpoint circle generation algorithm. Draw circle having center(0,0) and radius 9 using this algorithm. [11 Marks]

Midpoint circle generation algorithm ek efficient method hai circles ko draw karne ke liye. Yeh algorithm circle ke octants ko use karke circle ko generate karta hai.

Midpoint Circle Algorithm:

  1. Set the initial point (0, r) where r is the radius.
  2. Calculate the decision parameter p = 1 - r.
  3. For each point from 0 to r:
  4. If p < 0, plot the next point in the same direction.
  5. If p >= 0, plot the next point and adjust the decision parameter.

Is tarike se (0,0) center aur radius 9 ka circle generate kiya ja sakta hai.

Write Cohen Sutherland algorithm for Line clipping. [11 Marks]

Cohen-Sutherland algorithm line clipping ka ek efficient method hai jo lines ko clip karta hai based on a rectangular clipping window. Yeh algorithm regions ko divide karta hai aur har endpoint ke liye codes assign karta hai.

Cohen-Sutherland Algorithm:

  1. Define the clipping window with boundaries.
  2. Assign codes to each endpoint of the line.
  3. Use logical operations to determine the position of the line with respect to the window.
  4. Clip the line according to the codes assigned.

Is tarike se lines ko efficient clipping milti hai.

Explain flood filling and boundary filling methods. Give its two real-life examples where they are used. [11 Marks]

Flood filling aur boundary filling methods graphical regions ko fill karne ke liye use hoti hain.

Flood Filling:

Flood filling method ek area ko fill karne ka process hai starting from a given point. Yeh method recursively adjacent pixels ko fill karta hai jab tak filling ki area ko complete nahi kiya jata.

Boundary Filling:

Boundary filling method mein area ko fill karne ke liye boundary pixels ko define kiya jata hai. Is method mein filling ki shuruaat boundary se hoti hai aur baad mein area ko fill kiya jata hai.

Real-life Examples:

  • Digital image editing software (like Photoshop) uses flood filling to fill colors in drawings.
  • Computer-aided design (CAD) applications use boundary filling for area calculations in drawings.
